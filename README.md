# uWi-Fi
uWi-Fi is the decentralized Wi-Fi sharing platform using uPort.  
It helps to solve Wi-Fi connection probrem for foreign travelers to Japan.

## Overview
You can earn token by lending bandwidth to people who want to use Wi-Fi.  
It is like a mining using bandwidth resource.
